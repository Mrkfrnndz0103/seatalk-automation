Event: New Mentioned Message From Group Chat
Event Description
This event is triggered when a group member mentions the bot using '@' in a text message in a group chat. '@All' will not trigger this event.

Starting from 11 June 2024, each event will provide a thread_id to indicate whether the user has mentioned the bot in a thread. (Learn more about threading messages in group chats)
